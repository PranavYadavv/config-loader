# config-loader

Universal configuration loader for YAML, JSON, TOML and env files

## Installation

```bash
pip install config-loader
```

## Quick Start

```python
from config_loader import Config
client = Config()
client.run()
```

## Features

- High performance and memory efficient
- Comprehensive error handling
- Well tested with full coverage
- MIT licensed
